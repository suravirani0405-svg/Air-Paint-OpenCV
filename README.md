# Air-Paint-OpenCV
Air Paint using OpenCV is a real-time drawing app built with Python and OpenCV. It uses HSV color segmentation to track a blue pen cap through a webcam and lets users draw on a virtual canvas. It includes brush, eraser, and shape modes with adjustable size and colors, showcasing computer vision and interactive image processing.


## How to Run

1. Clone the repo:
   git clone https://github.com/suravirani0405-svg/suravirani0405-svg.git

2. Navigate to the folder:
   cd "air painter"   # or your project folder name

3. Install dependencies:
   pip install -r requirements.txt

4. Run the app:
   python virtual_painter.py


## Features
- Real-time drawing using webcam
- Brush, eraser, and shapes
- Adjustable colors and brush size
- Simple and interactive interface
